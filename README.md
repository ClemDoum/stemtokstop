README
======
stemtokstop = stemmer + tokenizer + remove stop words.  It's nothing special, but a simple application of NLTK wrapped with Flask.


INSTALL
=======
    $ pip install nltk
    $ python
    >>> import nltk
    >>> nltk.download() # And download all
    >>> ^D
    $ python stemtokstop.py
